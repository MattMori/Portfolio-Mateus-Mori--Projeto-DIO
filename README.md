# Portfolio - Mateus Mori

Este é o repositório do meu portfólio pessoal, que foi desenvolvido como parte do projeto da DIO (Digital Innovation One). O portfólio é uma página da web que exibe informações sobre mim, minhas habilidades, projetos, experiência profissional e treinamento.

## Funcionalidades

- Exibe informações sobre o meu perfil, incluindo nome, foto, cargo, localização, telefone e e-mail.
- Lista minhas habilidades técnicas e sociais.
- Mostra os idiomas que falo.
- Apresenta uma lista de projetos no meu portfólio, com links para cada projeto no GitHub.
- Detalha minha experiência profissional até o momento.
- Descreve meu treinamento acadêmico e cursos relacionados à área de desenvolvimento de software.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Estrutura do Projeto

O projeto está organizado em diferentes arquivos e diretórios:

- `index.html`: Arquivo HTML principal que contém a estrutura básica da página do portfólio.
- `css/`: Diretório que contém os arquivos CSS para estilizar a página do portfólio.
- `js/`: Diretório que contém os arquivos JavaScript para manipular os dados e atualizar a página.
- `data/profile.json`: Arquivo JSON que contém os dados do perfil.
- `imgs/`: Diretório que contém as imagens utilizadas no projeto, como fotos e ícones.
- `fonts/`: Diretório que contém as fontes adicionais utilizadas no projeto.

## Como Executar o Projeto

1. Clone o repositório em sua máquina local.
2. Abra o arquivo `index.html` em seu navegador da web.
3. A página do portfólio será exibida com todas as informações preenchidas com base nos dados do arquivo `profile.json`.

## Personalização

Para personalizar o conteúdo do portfólio com suas próprias informações e projetos, siga estas etapas:

1. Edite o arquivo `data/profile.json` para atualizar os dados do perfil com suas próprias informações.
2. Adicione suas próprias imagens no diretório `imgs/` e atualize os URLs das imagens nos arquivos HTML e CSS conforme necessário.
3. Personalize os estilos nos arquivos CSS localizados no diretório `css/` para atender às suas preferências de design.
4. Se necessário, adicione ou edite os arquivos JavaScript no diretório `js/` para ajustar o comportamento e a funcionalidade da página.
